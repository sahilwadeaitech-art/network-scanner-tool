# Changelog

## 1.2.0 (2024-12-15)

- added network health score to dashboard
- latency distribution stats
- scan history tracking (last 20 scans)
- port service breakdown analytics
- cleaned up report formatting

## 1.1.0 (2024-11-28)

- traceroute diagnostic tool
- DNS lookup with nslookup fallback
- tabbed diagnostics panel
- export report button in sidebar
- fixed progress bar not resetting after stop

## 1.0.0 (2024-11-10)

- initial working version
- network discovery (ping + TCP fallback)
- port scanner with banner grabbing
- ping diagnostic with parsed stats
- dark theme UI with CustomTkinter
- txt report generation
- dashboard with stat cards and activity log

## 0.2.0 (2024-10-22)

- switched to threaded scanning (way faster)
- added progress bar and device list
- MAC address detection from ARP table
- hostname resolution

## 0.1.0 (2024-10-08)

- basic project structure
- simple ping sweep scanner
- early UI prototype
